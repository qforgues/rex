QUICK REFERENCE GUIDE
================================================================================

WHAT DO I NEED TO KNOW?
  - Three credit card formats: AmEx, Chase, Visa
  - Detection via pattern matching on headers and distinctive features
  - Confidence scoring system (0-100%)
  - Organization into separate folders by type

WHAT ARE THE KEY COMPONENTS?
  - CardFormatDetector: Identifies format with confidence score
  - StatementOrganizer: Batch processes and organizes statements
  - Format mappings: JSON specifications for each card type

HOW DOES IT WORK?
  1. Read statement file
  2. Match against known patterns (headers, features)
  3. Calculate confidence score
  4. Assign to appropriate output folder
  5. Generate processing report

WHAT CONFIDENCE SCORES MEAN?
  95-100%: Confident identification, proceed with organization
  80-94%:  Good identification, can organize
  60-79%:  Possible match, review recommended
  < 60%:   Low confidence, move to Unknown folder

WHAT ARE DISTINCTIVE FEATURES?
  AmEx: Rewards points, business categories, multi-member
  Chase: Merchant locations, Ultimate Rewards
  Visa: Debit/Credit indicators, running balance

WHAT'S IN THE OUTPUT?
  - Organized statement folders (AmEx_Business, Chase_Personal, Visa, Unknown)
  - processing_results.json (detailed results)
  - Organization summary (human-readable)

CAN I ADD NEW CARD TYPES?
  Yes - create new JSON mapping, register in code, add output folder

USAGE EXAMPLES
================================================================================

BASIC USAGE:
  python3 multi_card_processor.py

WITH CUSTOM DIRECTORIES:
  Edit multi_card_processor.py main() function:
    input_directories = ["/your/path/1", "/your/path/2"]
    output_base = "/your/output"

LIBRARY USAGE:
  from multi_card_processor import CardFormatDetector
  detector = CardFormatDetector()
  sig = detector.detect_format("file.csv")
  print(sig.card_type, sig.confidence)

BATCH PROCESSING:
  from multi_card_processor import StatementOrganizer
  org = StatementOrganizer("/output/path")
  results = org.process_statements(["/input/dir1", "/input/dir2"])
  org.save_results("/output/results.json")
