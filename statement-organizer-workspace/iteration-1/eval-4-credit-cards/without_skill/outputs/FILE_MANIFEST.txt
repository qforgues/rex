================================================================================

To add support for additional card types:

1. Create new format mapping file:
   - Copy existing JSON template
   - Update identifier patterns
   - Define distinctive features
   - Specify field mappings

2. Register in CardFormatDetector:
   - Add to _load_mappings() mapping dictionary
   - New format automatically loaded

3. Update StatementOrganizer:
   - Add output folder mapping in _process_file()
   - Folder automatically created for new type

4. Test thoroughly:
   - Verify detection with sample files
   - Validate confidence scoring
   - Check output organization
