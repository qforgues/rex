================================================================================
                         5 DETECTION METHODS
================================================================================

1. STATISTICAL OUTLIERS (Z-Score)
   Flags: Amounts significantly outside normal range
   Example: $5,000 charge when your average is $50
   Result: Severity 1.0 (CRITICAL)

2. MERCHANT PATTERNS
   Flags: New or rarely-used vendors
   Example: First charge to "Crypto Exchange Platform"
   Result: Severity 0.6 (HIGH)

3. TIME-BASED ANOMALIES
   Flags: Transactions at unusual times
   Example: 3:22 AM charge when you normally shop 9am-3pm
   Result: Severity 0.35 (WARNING)

4. DUPLICATE DETECTION
   Flags: Identical transactions within 24 hours
   Example: Two identical $42.50 charges at Target
   Result: Severity 0.8 (HIGH)

5. FREQUENCY ANALYSIS
   Flags: Unusual concentration of high-value transactions
   Example: 25% of transactions are 2x your average
   Result: Severity 0.5 (ALERT)

================================================================================
                         HOW TO USE YOUR DATA
================================================================================

OPTION A: CSV File (Most Common)
  from anomaly_detection import TransactionAnomalyDetector
  import csv
  
  # Load historical baseline (3+ months)
  historical = []
  with open('statement_feb_2026.csv', 'r') as f:
      for row in csv.DictReader(f):
          historical.append({
              'date': row['Date'],
              'merchant': row['Merchant'],
              'amount': row['Amount'],
              'type': row['Type']
          })
  
  # Load current month
  current = []
  with open('statement_march_2026.csv', 'r') as f:
      for row in csv.DictReader(f):
          current.append({
              'date': row['Date'],
              'merchant': row['Merchant'],
              'amount': row['Amount'],
              'type': row['Type']
          })
  
  # Analyze
  detector = TransactionAnomalyDetector(historical)
  anomalies = detector.detect_anomalies(current)
  
  # View results
  for a in anomalies:
      if a['severity_score'] >= 0.7:  # Critical only
          print(f"CRITICAL: {a['reason']}")
  
  # Save report
  detector.export_json('march_2026_anomalies.json')

OPTION B: Manual Data Entry
