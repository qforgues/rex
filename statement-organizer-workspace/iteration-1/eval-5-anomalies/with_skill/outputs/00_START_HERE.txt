ANOMALY DETECTION IN 30 SECONDS
===============================

The skill uses 4 independent detection channels:

  1. AMOUNT DEVIATION
     Flags transactions significantly larger than your normal spending
     Example: You normally spend $400/month, a $5000 charge gets flagged

  2. MERCHANT PATTERN
     Flags new merchants and suspicious categories
     Example: First Amazon = LOW, first Crypto Exchange = HIGH

  3. DUPLICATE DETECTION
     Flags identical transactions (processing errors)
     Example: Three identical $459.99 charges = MEDIUM severity

  4. SPENDING PATTERN
     Flags unusual behaviors
     Example: 6 coffee runs in 1 hour = suspicious pattern

Each transaction is assigned a severity:
  - HIGH: Immediate investigation required
  - MEDIUM: Review recommended
  - LOW: Informational

THE ALGORITHM
=============

Amount Deviation (Statistical):
  Formula: Flag if amount > (mean + 2.5 * stdev)
  Confidence: 98.8% (3-sigma rule)
  Severity scales with deviation multiple

Merchant Pattern (Classification):
  Suspicious keywords: CRYPTO, GAMBLING, WIRE, etc.
  Known/unknown merchant tracking
  Auto-learns your spending patterns

Duplicate Detection (Exact Match):
  Same amount + same merchant + same day
  Indicates processing error or fraud testing
  Medium severity auto-flag

Pattern Analysis (Behavioral):
  Frequency analysis (multiple transactions per merchant)
  Time analysis (unusual hours)
  Escalation detection (increasing amounts)
