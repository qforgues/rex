================================================================================
MATCHING ALGORITHM AT A GLANCE
================================================================================

FOR EACH TRANSACTION:
  1. Extract: date, amount, merchant
  2. Search: Find all invoices with amount = transaction amount (±$0.01)
  3. If ZERO matches → UNMATCHED (no invoice for this amount)
  4. If ONE match → Calculate confidence with merchant name
     - If confidence ≥ 90% → MATCHED (auto-mark paid) ✓
     - If confidence 70-89% → AMBIGUOUS (needs user review) ⚠
     - If confidence < 70% → UNMATCHED (investigate) ❌
  5. If MULTIPLE matches → Use merchant name to pick best match
     - Compare "ABC WELLNESS" (bank) vs "ABC Wellness Collective" (FreshBooks)
     - Calculate similarity ratio (0.0 to 1.0)
     - Score = (100% amount + similarity% name) / 2
     - Apply thresholds as above

CONFIDENCE FORMULA:
  Confidence = (Amount Match Score + Merchant Name Similarity) / 2
  
  Example 1: Exact amount + exact name
    (100% + 100%) / 2 = 100% → MATCHED ✓
  
  Example 2: Exact amount + good name match
    (100% + 80%) / 2 = 90% → MATCHED ✓
  
  Example 3: Exact amount + weak name match
    (100% + 50%) / 2 = 75% → AMBIGUOUS ⚠
  
  Example 4: No matching amount
    (0% + N/A) / 2 = 0% → UNMATCHED ❌
