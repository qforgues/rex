### Output/Reference Files

#### `sample_mapping.json`
**Purpose:** Example of auto-generated format mapping
**Type:** JSON configuration file
**Size:** ~1.6 KB

**Structure:**
```json
{
  "bank_name": "NewBank",
  "format_name": "NewBank-CSV-Standard",
  "detection_confidence": 0.87,
  "fields": { ... },
  "required_fields": [ ... ],
  "validation_rules": { ... },
  "parsing_rules": { ... }
}
```

**Key Sections:**
- `fields`: Maps column names to field types with confidence
- `required_fields`: Must-have columns for parsing
- `validation_rules`: Regex patterns for validation
- `parsing_rules`: Date format, decimals, currency

**Use For:**
- Understanding format structure
- Manual customization of auto-detected formats
- Template for other banks
- Integration validation

**Generated By:** `FormatDetector.generate_format_mapping()`
